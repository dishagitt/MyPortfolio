About Me

I’m Disha Prajapati, an MCA graduate who enjoys building reliable, scalable web applications with a strong focus on the Python backend. I work with frameworks like Flask, FastAPI, and Django, design RESTful APIs, and implement secure authentication using JWT/OAuth.

I’m comfortable across the stack—structuring data models in MySQL, PostgreSQL and SQLite (with ORM tools like SQLAlchemy), and developing clean, responsive UIs with HTML, CSS, JavaScript, React, Bootstrap, and Tailwind. I focus on writing modular, efficient, and maintainable code while ensuring good performance and usability. During my academic journey, I’ve developed several projects that strengthened my understanding of API design, database architecture, and front-end integration.

Beyond technical development, I believe in collaboration and continuous growth. I enjoy working in team environments where I can share ideas, learn from others, and contribute to projects that have a meaningful impact. I also stay updated with modern development tools, testing practices, and deployment workflows using platforms like GitHub, Reddit, and Linkedin. I enjoy transforming complex ideas into interactive, user-friendly solutions that deliver real value to users, and I’m always curious to learn new frameworks, tools, and design patterns that can help me build better products and grow as a full-stack developer.
